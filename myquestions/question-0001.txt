Título del Problema: Predicción Base y Filtrado de Características Numéricas
El Contexto:
Trabajas como científico de datos en una empresa de bienes raíces y te han entregado un conjunto de datos "crudo" en un DataFrame de Pandas. Este conjunto contiene múltiples variables sobre las propiedades (como precios, metros cuadrados, número de habitaciones, etc.), pero también tiene columnas de texto libre, variables categóricas sin procesar y varias filas con datos faltantes (NaN).

Para establecer una línea base de predicción rápida (baseline), tu equipo ha decidido que, por ahora, ignorarán las columnas no numéricas e intentarán predecir el precio utilizando un árbol de decisión únicamente con los datos numéricos completos.

Tu Misión:
Implementa una función de Python que automatice el preprocesamiento básico, el entrenamiento de un modelo de árbol de regresión y el cálculo de la métrica de error (RMSE). La función debe realizar exactamente los siguientes pasos:

Filtrar datos numéricos (Pandas/NumPy): A partir del DataFrame original, quédate únicamente con las columnas que sean de tipo numérico (puedes apoyarte en Pandas y los tipos de datos de NumPy). Asume que la columna objetivo (target) es numérica y se mantendrá tras este filtro.

Limpiar nulos (Pandas): Elimina cualquier fila que contenga al menos un valor faltante (NaN) en el subconjunto de datos numéricos filtrado.

Separar variables: Divide el DataFrame resultante en la matriz de características X (todas las columnas numéricas excepto la objetivo) y el vector objetivo y.

Dividir los datos (Sklearn): Utiliza train_test_split para dividir X e y en conjuntos de entrenamiento y prueba. Debes usar el tamaño de prueba especificado en los argumentos y fijar el parámetro random_state=42 para asegurar la reproducibilidad.

Entrenar el modelo (Sklearn): Instancia y entrena un modelo DecisionTreeRegressor (fijando también su random_state=42) utilizando los datos de entrenamiento.

Evaluar y calcular RMSE (Sklearn/NumPy): Haz predicciones sobre el conjunto de prueba, calcula el Error Cuadrático Medio (MSE) y finalmente utiliza NumPy para calcular la raíz cuadrada de este valor, obteniendo así el RMSE.
