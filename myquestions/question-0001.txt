Problema 1: Detección Robusta de Anomalías con Envolvente Elíptica
El Contexto: Trabajas en el área de seguridad de una entidad financiera analizando transacciones que siguen una distribución aproximadamente normal (Gaussiana). A diferencia de otros métodos de detección de anomalías, tu equipo quiere usar la Envolvente Elíptica, que es más robusta para datos con distribución conocida pero contaminada con valores atípicos.
Tu Misión: Escribe una función que limpie los datos, ajuste el modelo y cuente cuántas anomalías se detectaron.
Recibe un DataFrame y un parámetro de contaminación.
Utiliza SimpleImputer de sklearn para rellenar nulos con la media.
Instancia y entrena EllipticEnvelope de sklearn.covariance con el parámetro contamination y un random_state=42.
Utiliza el modelo para predecir las etiquetas (1 para normal, -1 para anomalía).
Utiliza numpy para contar cuántas filas fueron marcadas como -1.
Nombre de la función: conteo_anomalias_elipticas
Argumentos: df (pd.DataFrame), contaminacion (float)
Objeto a devolver: int (el número total de anomalías encontradas).
